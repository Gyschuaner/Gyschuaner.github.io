# RPG待办事项应用

一个融合了RPG养成元素的待办事项应用，通过完成任务获得经验值，提升等级，解锁成就！

## 功能特点

### 📝 待办事项管理
- 添加不同难度的任务（简单、中等、困难）
- 标记任务完成状态
- 删除不需要的任务
- 筛选查看全部/未完成/已完成任务

### ⚔️ RPG养成系统
- **经验值系统**：完成任务获得经验值
  - 简单任务：+10 经验值
  - 中等任务：+25 经验值
  - 困难任务：+50 经验值
- **等级进阶**：累积经验值提升等级
- **属性成长**：每次升级获得全属性+1
  - 力量 (⚔️)：代表你的完成任务能力
  - 防御 (🛡️)：代表你对抗拖延症的能力
  - 智力 (🧠)：代表你的任务管理智慧

### 🏆 成就系统
- 初出茅庐：完成第一个任务
- 任务达人：完成10个任务
- 升级高手：达到5级
- 经验猎手：累计获得500经验值
- 挑战大师：完成5个困难任务

### 💾 数据持久化
- 所有数据保存在本地存储中
- 刷新页面不会丢失进度

## 如何使用

1. 打开 `index.html` 文件即可开始使用
2. 在输入框中输入任务内容
3. 选择任务难度（影响获得的经验值）
4. 点击「添加」按钮创建任务
5. 完成任务时勾选复选框获得经验值
6. 查看你的等级、经验值和属性成长
7. 解锁成就获得额外成就感

## 游戏机制

### 等级计算公式
- 初始等级：1级
- 初始经验值需求：100
- 每升一级，下一级所需经验值增加50%

### 升级收益
- 每次升级获得全属性+1
- 等级提升时会显示庆祝动画

### 成就解锁
- 达成条件后自动解锁成就
- 解锁时会显示成就通知
